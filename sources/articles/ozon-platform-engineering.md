# Ozon Tech — Platform Engineering

Practical tutorial from Ozon Tech on database sharding as a core platform engineering concern. Walks through client-side sharding of PostgreSQL using Go, covering shard key selection, parallel query execution across shards, migration scripting, and resharding strategies. Demonstrates how internal platform teams tackle horizontal scaling.

## Key Topics

- Client-side sharding of PostgreSQL
- Shard key selection and data distribution
- Parallel read/write across shards in Go
- Migration and resharding strategies
- Platform engineering for developer experience
- CI/CD considerations for sharded databases
