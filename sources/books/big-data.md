# Big Data: Principles and Best Practices of Scalable Real-Time Data Systems

**Автор:** Nathan Marz, James Warren
**Год:** 2015

Представляет Lambda Architecture — подход к построению Big Data систем, сочетающий batch layer (неизменяемое хранение, предвычисление views), serving layer (индексация batch views) и speed layer (real-time обработка, компенсация задержки). Подробное описание Data Model for Big Data, принципов неизменяемости данных, человеко-читаемых рекомбинаций, архитектуры SuperWebAnalytics.com.

## Практическая ценность

Основа для понимания архитектур больших данных. Lambda и Kappa architectures — recurring темы в patterns/architecture-design/.
