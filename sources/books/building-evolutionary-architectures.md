# Building Evolutionary Architectures

**Автор:** Neal Ford, Rebecca Parsons, Patrick Kua, Pramod Sadalage
**Год:** 2022
**Издание:** 2nd

Вводит концепцию evolutionary architecture — архитектуры, которая поддерживает incremental change как первоклассный атрибут. Ключевые идеи: guided by fitness functions (автоматизированные проверки архитектурных характеристик), incremental change как принцип, архитектурные кванты, last responsible moment. 2-е издание добавляет непрерывное соответствие требованиям, эволюцию архитектуры в мэйнфреймах/микросервисах.

## Практическая ценность

Источник для patterns/ — разделы по fitness functions, архитектурной эволюции, continuous architecture. Fitness functions используются как практический инструмент валидации архитектуры.
