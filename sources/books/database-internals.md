# Database Internals: A Deep-Dive into How Distributed Data Systems Work

**Автор:** Alex Petrov
**Год:** 2019

Практический разбор внутреннего устройства баз данных и storage engines. Часть 1: Storage Engines (B-Trees, LSM Trees, Buffer Pool, Page Cache, WAL, Compression). Часть 2: Distributed Systems (Failure Detection, Leader Election, Replication, Distributed Transactions, Consensus). Объясняет, почему разные БД выбирают разные структуры данных и распределённые алгоритмы.

## Практическая ценность

Незаменим для понимания выбора БД — объясняет, что происходит под капотом. Используется в patterns/fundamentals/ и tools/databases/.
