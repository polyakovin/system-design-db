# Design Patterns: Elements of Reusable Object-Oriented Software

**Автор:** Erich Gamma, Richard Helm, Ralph Johnson, John Vlissides (Gang of Four)
**Год:** 1994

Канонический каталог 23 паттернов объектно-ориентированного проектирования, разделённых на три группы: порождающие (Abstract Factory, Builder, Singleton и др.), структурные (Adapter, Composite, Proxy и др.) и поведенческие (Observer, Strategy, Template Method и др.). Каждый паттерн описан по схеме: Intent, Motivation, Applicability, Structure, Participants, Collaborations, Consequences, Implementation, Sample Code.

## Практическая ценность

Фундаментальный источник для понимания паттернов, на которые опираются более высокоуровневые архитектурные решения. Многие паттерны (Observer, Strategy, Facade) эволюционировали в архитектурные стили.
