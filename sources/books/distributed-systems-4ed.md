# Distributed Systems (4th Edition)

**Автор:** Maarten van Steen, Andrew S. Tanenbaum
**Год:** 2023
**Издание:** 4th
**ISBN:** 9789081540636

Учебник по распределённым системам: архитектуры (включая архитектурные стили, middleware, организация), процессы (threads, virtualization, clients, servers, code migration), коммуникация (RPC, RMI, Message-Oriented, Multicast), именование (flat, structured, attribute-based), синхронизация (clock sync, logical clocks, mutual exclusion, election, consensus), консистентность и репликация, отказоустойчивость, безопасность.

## Практическая ценность

Академический фундамент для понимания распределённых систем. Разделы по консенсусу, репликации и согласованию времени используются в patterns/.
