# Documenting Software Architectures: Views and Beyond

**Автор:** Paul Clements, Felix Bachmann, Len Bass, David Garlan, James Ivers, Reed Little, Paulo Merson, Robert Nord, Judith Stafford
**Год:** 2010
**Издание:** 2nd

Практическое руководство по документированию архитектуры ПО. Вводит концепцию architectural views/viewtypes (module, component-and-connector, allocation), стили для каждого viewtype, шаблоны документирования интерфейсов и поведения, документирование вариабельности и динамических систем. Методология SEI для создания полезной и удобоваримой архитектурной документации.

## Практическая ценность

Источник стандартов документирования. Определяет, какие viewtypes и стили использовать для разных стейкхолдеров. Полезен при структурировании знаний о системе.
