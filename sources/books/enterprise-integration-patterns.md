# Enterprise Integration Patterns: Designing, Building, and Deploying Messaging Solutions

**Автор:** Gregor Hohpe, Bobby Woolf
**Год:** 2004
**ISBN:** 0321200683

Каталог 65 паттернов интеграции приложений через message-oriented middleware. 9 категорий: Integration Styles, Channel Patterns, Message Construction, Routing, Transformation, Endpoint, System Management. Каждый паттерн — proven solution с примерами. Язык паттернов остаётся актуальным для cloud integration и IoT.

## Практическая ценность

Базовый источник для паттернов интеграции: Message Queue, Publish-Subscribe, Dead Letter Channel, Competing Consumers и др. Используется в patterns/architecture-design/ для разделов по messaging.
