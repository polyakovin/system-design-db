# Fundamentals of Software Architecture: An Engineering Approach

**Автор:** Mark Richards, Neal Ford
**Год:** 2020
**Издание:** 1st

Первый comprehensive обзор software architecture как инженерной дисциплины. Систематизирует архитектурные характеристики (availability, elasticity, scalability, и др.), вводит архитектурные стили (layered, pipeline, microkernel, service-based, event-driven, microservices), четыре квадранта измерения архитектуры. Даёт практические инструменты: architecture decision records, fitness functions, анализ trade-offs.

## Практическая ценность

Ключевой источник для patterns/ — определения архитектурных стилей, характеристик, fitness functions. Architecture decision records используются как стандарт документирования.
