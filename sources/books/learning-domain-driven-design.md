# Learning Domain-Driven Design: Aligning Software Architecture and Business Strategy

**Автор:** Vlad Khononov
**Год:** 2021

Практическое введение в DDD для современных разработчиков. Часть 1: Strategic Design (Bounded Context, Ubiquitous Language, Subdomains — Core/Supporting/Generic, Context Mapping). Часть 2: Tactical Design (Aggregates, Domain Events, Value Objects, Domain Services, Repositories). Часть 3: DDD и другие методологии (Microservices, Event Sourcing, CQRS, Data Mesh). Реальный кейс внедрения DDD в стартапе.

## Практическая ценность

Современный и практичный взгляд на DDD. Используется в patterns/ — разделы по bounded context, event storming, стратегическому и тактическому дизайну.
