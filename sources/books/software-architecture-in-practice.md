# Software Architecture in Practice

**Автор:** Len Bass, Paul Clements, Rick Kazman
**Год:** 2021
**Издание:** 4th

Фундаментальный учебник от SEI (Software Engineering Institute). Вводит понятие architecture as a vehicle for stakeholder concerns, quality attribute scenarios, архитектурные тактики для каждого quality attribute (availability, modifiability, performance, security, testability, usability), методы анализа архитектуры (ATAM, CBAM). 4-е издание добавляет DevOps, continuous architecture, архитектуру для AI-систем.

## Практическая ценность

Базовый учебник для patterns/fundamentals. Определяет язык описания архитектур и quality attributes. Тактики из книги разнесены по соответствующим разделам patterns/.
