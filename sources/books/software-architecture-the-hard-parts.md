# Software Architecture: The Hard Parts

**Автор:** Neal Ford, Mark Richards, Pramod Sadalage, Zhamak Dehghani
**Год:** 2021

Книга о сложных trade-offs в distributed architectures, для которых нет best practices. Через историю команды Sysops Squad рассматривает: granularity сервисов, управление workflow и orchestration, управление и decoupling контрактов, распределённые транзакции (Saga, eventual consistency), операционные характеристики (scalability, elasticity, performance).

## Практическая ценность

Основной источник для patterns/architecture-design/ — разделы по сервисной гранулярности, управлению транзакциями, trade-off analysis.
