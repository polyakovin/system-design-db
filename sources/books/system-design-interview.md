# System Design Interview – An Insider's Guide (Volume 1)

**Автор:** Alex Xu
**Год:** 2020
**ISBN:** 9798664653403

Практическое руководство по подготовке к system design interview. 12 реальных сценариев проектирования (Design Twitter, YouTube, Google Drive, etc.) с пошаговым разбором: от определения требований до deep-dive в ключевые компоненты. Каждый кейс проходит 4 этапа: понимание проблемы, high-level design, deep dive, wrap-up.

## Практическая ценность

Основной источник разобранных кейсов для раздела patterns/fundamentals и patterns/architecture-design. Используется как reference для практических примеров.
