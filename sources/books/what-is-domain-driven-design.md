# What Is Domain-Driven Design?

**Автор:** Vlad Khononov
**Год:** 2019

Лёгкое введение в DDD для разработчиков, которые хотят понять суть, не читая 600-страничный труд Эванса. Объясняет: что такое domain model и ubiquitous language, strategic design (bounded context, context map, subdomains), tactical design (aggregates, domain events, value objects). Меньше деталей, чем Learning DDD, но быстрее вводит в курс дела.

## Практическая ценность

Быстрый старт для тех, кто не знаком с DDD. Введение в темы, которые детально раскрыты в patterns/.
