# System Design Blogs & Channels

Подборка блогов, YouTube-каналов и Telegram-ресурсов по системному дизайну и архитектуре ПО.

## Блоги и сайты

1. **System Design One** — https://systemdesign.one/categories/
   Разборы архитектур популярных сервисов: Slack, Zoom, Discord, WhatsApp и др. Deep-dive в реальные системы.

2. **ByteByteGo** — https://bytebytego.com/
   Известный ресурс по System Design: книги, шпаргалки, визуальные разборы. Создан Alex Xu (автор System Design Interview).

3. **ByteByteGo Blog** — https://blog.bytebytego.com/
   Статьи про архитектуру, техтреды, лучшие практики. Платформа Substack.

4. **Владислав Всеволодович** — https://vvsevolodovich.dev/
   Блог по системному дизайну на русском языке. Разборы архитектурных паттернов и подходов.

5. **nvashanin (Medium)** — https://medium.com/@nvashanin
   Статьи по System Design, pattern languages, архитектурным подходам. На русском и английском.

## YouTube-каналы

6. **ByteByteGo (YouTube)** — https://www.youtube.com/@ByteByteGo/videos
   Видео-разборы системного дизайна, визуализация архитектур, анимированные объяснения.

7. **Code of Architecture** — https://www.youtube.com/@FrontEndEngineer/videos
   Видео по архитектуре ПО и System Design. Канал на русском языке.

## Telegram

8. **Architecture Weekly** — https://t.me/architectureweekly
   Еженедельная подборка материалов по архитектуре ПО: статьи, доклады, инструменты.
