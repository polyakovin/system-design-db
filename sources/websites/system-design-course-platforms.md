# System Design Course Platforms

Подборка платных курсов и платформ по system design, OOD и смежным темам. Все источники — discovery-ссылки (статус `referenced`), контент не парсится.

## Курсы

### Grokking the System Design Interview

- **Платформа:** DesignGurus.io (ex-FAANG авторы)
- **URL:** <https://www.designgurus.io/course/grokking-the-system-design-interview>
- **Формат:** платный курс
- **Содержание:** оригинальная методология разбора system design задач, фокус на trade-offs, пошаговый framework для interview раундов.

### Grokking the Object-Oriented Design Interview

- **Платформа:** DesignGurus.io
- **URL:** <https://www.designgurus.io/course/grokking-the-object-oriented-design-interview>
- **Формат:** платный курс
- **Содержание:** OOD-паттерны, проектирование классов и интерфейсов для технических интервью.

### Educative.io System Design

- **Платформа:** Educative.io
- **URL:** <https://www.educative.io/search?skills=system-design>
- **Формат:** платформа с платными курсами
- **Содержание:** Grokking Modern System Design Interview и другие курсы по system design, распределённым системам и архитектуре.

### Karpov.Courses System Design

- **Платформа:** Karpov.Courses
- **URL:** <https://karpov.courses/systemdesign>
- **Формат:** платный видеокурс (русскоязычный)
- **Содержание:** практический курс по system design на русском языке с разбором архитектурных кейсов.

### Stanford Databases: Relational Databases and SQL (edX)

- **Платформа:** edX / Stanford University
- **URL:** <https://www.edx.org/learn/databases/stanford-university-databases-relational-databases-and-sql>
- **Формат:** платный курс
- **Содержание:** академический курс по реляционным базам данных и SQL от Stanford.

## Практическая ценность

- Охватывают system design, OOD и базы данных — три ключевые области interview preparation.
- DesignGurus-курсы задают стандарт методологии разбора system design задач.
- Русскоязычный курс полезен для тех, кому нужен материал на русском.
